⚡ Common Quick Fixes

🔋
Battery draining fast
Go to Settings → Battery → disable Background App Refresh and reduce screen brightness.
🐌
Phone running slowly
Clear cache: Settings → Storage → Cached Data → Clear. Then restart device.
📶
Wi-Fi not connecting
Forget the network and reconnect. If that fails, reset network settings.
📱
App keeps crashing
Update the app in Play Store. If still crashing, uninstall and reinstall.
